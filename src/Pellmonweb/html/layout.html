This is a demo of the PellMon webinterface. The demo database holds real logged data for 6 months, view past data by clicking the left arrow button below the graph. The demo is running disconnected for obvious reasons, so the database is not updated and the parameter values are fake. You can access the parameter values with user 'testuser' and password '12345'. The parameter view shows the data/parameters made available by the scottecom plugin, which handles communication with a NBE scotte/woody/biocomfort pellet burner, but PellMon also supports generic pellet burners and the functionality can be extended by adding a new plugin. Full source code is available at github
